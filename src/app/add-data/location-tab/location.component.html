Select and upload files from your computer. Property editing is enabled upon successful upload. Acceptable file types are .xls, .xlsx and .csv.

Ensure that your file name is no more than 64 characters in length and your data is listed only on the first tab.

Latitude and Longitude coordinates must be in decimal degrees format with no spaces and must also be formatted as numbers in your spreadsheet.

Header rows must be included:

Latitude and Longitude for coordinate files OR

Street Address, City, State and Zip Code for address files.

For performance reasons, there is a 500 MB file size limit for data uploads.
