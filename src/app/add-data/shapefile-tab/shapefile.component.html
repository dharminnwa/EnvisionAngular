When uploading shapefiles, be sure to include the .shp, the .shx, the .dbf and the .prj files.

ENvision data is properly viewed using the WGS 1984 Geographic Coordinate System, Datum, and Spheroid; with no Projected Coordinate System. If your data is in another format or coordinate system, it is not compatible with ENvision and must be converted for accuracy. If you need assistance adjusting the projection of your file, please contact MAPSearch support at help@mapsearch.com

For performance reasons, there is a 500 MB file size limit for data uploads.
