Select and upload files from your computer. Property editing is enabled upon successful upload.

If you are unable to successfully load your KML/KMZ file due to size or format, please reach out to us at help@mapsearch.com and we can attempt to convert your file to a shapefile.

For performance reasons, there is a 2 MB file size limit for KML uploads and a 1.5 MB file size limit for KMZ uploads.
